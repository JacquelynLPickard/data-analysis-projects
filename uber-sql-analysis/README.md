# Uber SQL Analysis
# Last week, I wrapped up a SQL project using a Kaggle Uber database — and it was great!

The dataset is built around realistic ride-sharing business scenarios, which made it the perfect environment to work through some advanced SQL concepts:

- Surge pricing analysis
- Driver churn & retention
- Customer cohort analysis
- Window functions & CTEs

What makes this dataset stand out is how well it mirrors real-world data relationships — the kind of messy, layered problems you actually encounter on the job.

If you're looking for a SQL project that goes beyond the basics and gets into real business problem-solving, this one is worth your time.

My Kaggle Notebook: https://lnkd.in/eNiTXpTD
Dataset: https://lnkd.in/ef8rxDfd
